「とにかく、この過去問に答えられるようになるための情報から調べよう……」

　きっとこれらの問いは、過去に誰かが経験したトラブルにまつわる問題なのだろう。
わたしは一つ一つ、参考書をあちこち探し回りながら問題解決を試みていった。

　＊

　《飛竜》とは、《赤嵐断層》以降に突如地球に現れた生物です。
空想上の《ドラゴン》と酷似した姿をしており、高い知性と飛行能力を持ちます。また、《赤嵐断層》発生年に観測された《赤い竜》との関連性を示す研究が行われていますが、彼らの生態は多くが謎のままです。
化石燃料が輸送エネルギーの要であった時代と《赤嵐断層》以降を分けて、前者は〈旧時代〉、後者を現代と呼びます。

◯《飛竜》の特徴・分類
　《飛竜》には三種の竜種が存在します。
それぞれ能力も外見的特徴も大きく異なりますが、その遺伝子情報はまったく一致しており、なぜこのような個体差が生まれるのかは謎のままです。
また、この地球環境において《飛竜》の自重をもつ生物が飛行するのは物理的に矛盾しており、なんらかの未知の力が働いていることが考えられています。

ワイバーン級（L級）：
　体長約20メートル程度（翼長、尾も含めて）、体重約850kg～1トン程度。
　小型の竜種。高速飛行を可能とし、小回りの効く運用に向く。〈旧時代〉のヘリコプター、自動車の運用に相当。
　
ドラゴン級（D級）：
　体長約50メートル程度（翼長、尾も含めて）、体重約10トン程度
　中型竜種。長距離飛行、より多くの貨物が可能。〈旧時代〉の大型トラック、バスの運用に相当。

タラスク級（T級）：
　体長約100メートル程度、体重約100トン程度。
　翼が退化し飛行能力を失った代わりに、超巨大な胴体を持ち、大量の人と物資を乗せられる。〈旧時代〉の鉄道・輸送船の運用に相当。
　現在、日本において三頭のタラスク級を所有しており、東・西日本、北海道間の重要な輸送を担っています。

◯《飛竜》の運用
　《飛竜》は高い飛行能力を持ち、地球上のどの生物や〈旧時代〉の飛行機よりも安全で安定した運航が可能です。まさに空の絶対王者といえます。
しかしそれはあくまで彼らの自重で飛行する場合で、人間や貨物を積んだ場合はその限りではありません。急激な環境変化（風の急変（ウインドシア）、突風前線、雷雨接近など）との遭遇は致命的になり得る可能性があります。
　離着陸する自治体の竜港管制圏内では、飛行禁止の指示が出た場合は直ちに最寄りの竜港に着陸、もしくはエアハブに避難することが義務付けられています。
また、竜種ごとに設定された飛行続行指数WCI（Wyvern Continuation Index）を厳守し、定期的に計器による測定を行い、安全飛行が確保されることを義務付けられています。

◯積載率LR
　積載率 LR = W_payload(積載量kg) / W_body(飛竜の体重kg)
　安定運航域の上限：LR_max = 0.25
　余裕（0〜1）：R = clamp(1 - LR / LR_max, 0, 1)

例：850kg個体の場合
　LR_max=0.25 → 安定積載上限：約212kg
　　積載150kg → LR=0.176 → R≈0.30（“余裕は残るが、荒れに弱い”）
　　積載210kg → LR=0.247 → R≈0.01（“限界域、荒れたら即退避”）

◯続行指数
入力（10分ごとに更新）
　ΔV_vec：10分間の風の変化（風速＋風向の“ベクトル差”）
　ΔP：10分間の気圧変化
　GF：ガスト幅（V_gust - V）

ベース許容（空荷〜軽荷の標準）：
　T_w0 = 8 kt / 10min
　T_p0 = 2 hPa / 10min
　T_g0 = 12 kt

余裕R：
　T_w(R) = T_w0 * (0.3 + 0.7R)
　T_p(R) = T_p0 * (0.3 + 0.7R)
　T_g(R) = T_g0 * (0.3 + 0.7R)

指数：
　WCI = sqrt( (ΔV_vec/T_w(R))^2 + (ΔP/T_p(R))^2 + (GF/T_g(R))^2 )

GREEN：WCI < 0.7
→ 続行OK（ただし10分ごと監視継続）
AMBER：0.7 ≤ WCI < 1.0
→ 続行は可だが (a)最寄り竜港へ収束 (b)高度をローカル層へ (c)積載が重いなら投棄/分割検討
RED：WCI ≥ 1.0
→ 続行NG。着陸待機 / 引き返し / 安全空域へ退避（優先）

　＊

　前半の内容はファンタジー小説みたいでぐっと興味がわいたものの、計算式が登場した瞬間に脳が拒否してしまう。
すごく大事なことだというだけは理解できる。

「これ機械が教えてくれんのかな……覚えなあかんの？」
「機械だって絶対じゃない。故障して動かんようになった時に、空で判断ができるのは人間だけだ」

　とつぜん頭の上から誰かの声がした。
伏せていた顔を上げ、声の主を見上げると、制服姿の少女が立っていた。

「志摩さん？」

　同じ中学のクラスメイト、志摩由理奈（しま　ゆりな）だった。
おさげに眼鏡という、まさにザ・委員長というステレオタイプ通りの姿。実際、彼女が学級委員であり生徒会長でもある。

「空知さん、ライセンス受験をするのか？」
「あ、うん。五月の試験に挑戦しよう思って……」
「だったら真剣に取り組んでほしい。車や飛行機と違って《飛竜》は生きとるから。何かトラブルが起きた時に、危険なのはあなただけではない。そして彼らは必死で人間を守ろうとする」

　志摩さんはまっすぐと見つめてそういった。否定はせず、ただ有無をいわせない切実さが感じられた。

「志摩さん、《飛竜》に詳しいん？」
「両親が管制官なんだ。私もいずれ《飛竜》に関わる仕事に就きたい。《飛竜》の乗り手――《操竜士》はまだまだ少なく、ライセンスを持っていれば引く手数多だろう。そのかわり、とても危険な仕事だ」
「危険……」

　落ちれば無事では済まない空を飛ぶのだ。当然わかってはいる。
しかし、改めて指摘されると心臓がきゅっとした。

「うん、覚悟はしとるつもり。でもありがとう、教えてくれて」
「……《操竜士》は空を飛ぶことだけがわかっていればいい。生態や歴史といった《飛竜》と暮らすサポートは別の職業でやることだから。試験では、飛行中の判断の基礎や、飛行計画の作成問題などが出題される。実践的な問題に答えられていればパスできる」
「！　ありがとう、わたしがんばる！」
「ああ……」

　そう、志摩さんはそっけなく返事だけすると、言うべきことを言って気が済んだ様子で、すたすたと去っていった。

（生きている、か。志摩さん、《飛竜》のことがほんとうに好きなんやな）

　つい自分の目標のことだけを意識してしまいがちだったが、犬や猫などのペットと同じ。いや、命を預け合うのだから対等以上だ。
自分の過ちで迷惑をかけるどころか、命を落とすことになるだなんて耐えられない。
　わたしは、気を引き締めなおして、再び問題に取り組むのだった。

　＊

　そしてあっという間のニヶ月。
わたしは三年生になり、受験資格である一五歳の誕生日を超えた。
迎えた試験当日。会場は高松にある香川県庁の大会議室に設営される試験場だ。
　土庄港から一日に何便か出る《飛竜》の送迎便を待っていると、レナとスバルが空から降りてきた。

「や、久しぶり。行くんだね」
「うん」
「乗せてったるわ。JC二人なんて、スバルには楽勝やで」

　レナが気前よく言うと、スバルは「クア」と返事をした。なんとなくだが、この鳴き声は肯定を意味しているようだ。

「ありがとう。わたし、緊張してて、話し相手欲しかってん」
「おけおけ、まかせて。どういうルートをとるかわかるよね？　飛行レグ（連続飛行）は通常１レグあたり六十分以内やけど、海上は制限がかかるよ。瀬戸内海は内海やけん、二十分」
「うん、豊島を経由して、途中の島で休憩を取りながら。やんな？」
「いいね。それが理解できとれば、あとは試験前にしっかりトイレに行っとけばバッチリよ」
「ふふ、元気づけてくれてありがとう。スバルもよろしくね？」

　レナとスバルは、同時に目を細めてにやっと笑った。（スバルは笑ったような気がした）
レナの腕で引き上げられて、わたしはスバルに騎乗した。ドラゴン級のエアバスには何度か乗ったことがあるが、ワイバーン級に二人乗りは初めてだ。
手伝ってもらいながらハーネスを装着し、貸してくれたヘルメットを被る。《飛竜》の飛ぶ高さから落ちた日には、ヘルメットなど無意味なのだが飛行中の風をもろに受けると、とても息を吸えたものじゃないので必要なのだ。

　「じゃあ掴まっとってな。　〈レクト！〉」
レナの掛け声にスバルが応え、すぐに重力を失ったような感覚でふわりと浮き上がる。

「……いってきます」

　今日中に小豆島に帰ってくるのだが、なんとなくそう呟いた。
あっという間に空が近くなり、わたし達は海の上を飛びたった。

　＊

　試験の手応えは、正直分からなかった。
　例の"安全指数"に関する問題も出題されていたが、ざっくりとした選択形式だった。
計算機としてソナピを持ち込むのは許可されていた（〈旧時代〉の遺物の「機内モード」にする条件で）ので、計算式をすべて登録していたのだが、使うことはなかった。
そして、配点が高かったのは志摩さんが言っていたとおり、不測の事態に対してどう対処するかの記述式問題だった。
覚えたところも出たけれど、全く知らない事態に関する問題もあって、それにはかなり焦った。わたしならどうするかを、自分なりに考えて全問埋めることはできたが…。
とにかく、できることはやり切った。そう思える。
　試験を終えて高松築港に戻ると、待ってくれていたレナが手を振っていた。

「さっぱりした顔しとるね。手応えあったん？」
「わかんない。でも、やることやったし、どんな結果でも悔いはない……。ううん、やっぱり合格したいわ」
「大丈夫でしょ、あたしが過去問あげたんやけん。……ま、落ちてもまた次受ければいいやん。生きている限り何度もチャンスあるわ」

　レナなりの励ましなのだろう。

「うん。でも、今回はそういうんじゃないんよ」

　ここで軽く受け流せるような自分なら、一生何やったってうまくいかない。
帰りは、行きよりも口数が少ないフライトとなった。

　＊

　結果発表は、一週間後。紙の郵送が昔より難しくなった今、重要な知らせもすべてSNSで済まされるようになっている。
わたしは、学校の昼休み、教室の隅でソナピの画面を凝視していた。
焦っても仕方がないとわかっていても、何度も更新ボタンを押して確認してしまう。Wi-Fiの電波が弱いのか、気が急いでいるからなのか、読み込み中のグルグルが長い。
　お弁当を食べながらぽちぽちしていた、その時、DMのリストに新しい通知がポンという音とともに追加された。
ごくり、と喉が鳴る。震える指でタップすると、全文の画面に切り替わった。
　わたしの受験番号、B-1042の文字とともに「合格」の二文字。
その意味を理解するより先に、心臓が大きく跳ねた。

「……受かった」

　声が出た。喜びというより、安堵が全身を駆け巡って、力が抜ける。
受かった。学科は通った。これで、《飛竜》に乗れる。
いや、まだだ。次は実技実習がある。
　でも、《操竜士》への道は通じたのだ。

『アリナ！　どうだった！？』

　レナからのメッセージが届く。

『やったよ！　合格しとった！』
『やったー！　おめ！！！！！　ようこそ空へ🐲』

　画面の向こうで、レナがニカっと笑っているのが見えるようだ。
わたしはソナピを胸に抱きしめて、大きく息を吸い込んだ。
窓の外には、緋い空が広がっている。今までなんだか寂しいと感じていた空が、昨日までとは、少しだけ色が違って見えた。
　わたしはあそこへ行くんだ。あの一角に、自分の居場所を見つけに行くんだ。
遠くで、飛竜の鳴き声が聞こえた気がした。それは、わたしに「はやくおいで」と呼ぶ声のようにも聞こえた。
　教習開始まで、あと一週間。
わたしの本当の夏が、始まろうとしていた。

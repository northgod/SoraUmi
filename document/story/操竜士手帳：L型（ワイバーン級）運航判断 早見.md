# 操竜士手帳：L型（ワイバーン級）運航判断 早見（v1.0）

## 1) 体格（自重）レンジ：物語で使いやすい基準

* **標準成体（基準値）**：**850 kg**（推奨）
* **軽量個体**：600 kg（敏捷・荒天に弱い）
* **大型個体**：1,100 kg（安定・維持コスト高）

> 本書の計算は「基準値 850 kg」を標準例とする。

---

## 2) 積載テンプレ（現場で使える内訳）

### 2-1. 標準搭乗（1名・操竜士）

* 操竜士（体重＋防寒着）…… **65〜85 kg**
* 鞍・ハーネス・保安装備 …… **18〜30 kg**
* 端末・無線・ログ機材 ……… **2〜5 kg**
* 水・非常食・救急 …………… **4〜8 kg**
* 脚装具（防疫ブーツ等） …… **3〜6 kg**

**合計（標準）**：**92〜134 kg**

### 2-2. 小荷物（生活飛行・近距離）

* 小型貨物・手荷物 ………… **0〜30 kg**

**合計（標準＋小荷物）**：**92〜164 kg**

### 2-3. 荷運び（無理のない上限を“安定運航域”に置く）

* 追加貨物（荷箱・袋） …… **30〜80 kg**（ここから荒天耐性が目に見えて落ちる）

**合計（標準＋追加貨物）**：**120〜240 kg**

---

## 3) 積載率LR（Load Ratio）と余裕R（Reserve）

### 3-1. 定義

* **積載率**：`LR = W_payload / W_body`
* **安定運航域の上限**：`LR_max = 0.25`（推奨：L型の安全運用ルールとして採用）
* **余裕（0〜1）**：`R = clamp(1 - LR / LR_max, 0, 1)`

> 直感：**LRが増えるほど、余裕Rが削れる。**
> 「飛べるか」ではなく「荒れたときに耐えられる余裕があるか」を数値化する。

### 3-2. 850kg個体の目安

* `LR_max=0.25` → 安定積載上限：**約212 kg**
* 例：積載150kg → `LR=0.176` → `R≈0.30`（“余裕は残るが、荒れに弱い”）
* 例：積載210kg → `LR=0.247` → `R≈0.01`（“限界域、荒れたら即退避”）

---

## 4) まず先に：計算より優先の「絶対NG（即中止/退避）」

WCI以前に、次が1つでも当たれば**続行しない**（航空の安全思想）。

* 雷雨・積乱雲の接近、ガストフロント疑い
* ウインドシア疑い（風向の急反転・風の急増減）
* 視程・雲底が最低基準未満（作品側のルールで固定）
* 竜港・エアハブから「航路層クローズ」「警報」発令

---

## 5) 続行指数 WCI（Wyvern Continuation Index）

### 5-1. 入力（10分ごとに更新）

* `ΔV_vec`：10分間の風の変化（風速＋風向の“ベクトル差”）
* `ΔP`：10分間の気圧変化
* `GF`：ガスト幅（`V_gust - V`）

### 5-2. 余裕Rで許容値を縮める（L型の“荷物に弱い”を反映）

ベース許容（空荷〜軽荷の標準）：

* `T_w0 = 8 kt / 10min`
* `T_p0 = 2 hPa / 10min`
* `T_g0 = 12 kt`

余裕でスケール：

* `T_w(R) = T_w0 * (0.3 + 0.7R)`
* `T_p(R) = T_p0 * (0.3 + 0.7R)`
* `T_g(R) = T_g0 * (0.3 + 0.7R)`

指数：

* `WCI = sqrt( (ΔV_vec/T_w(R))^2 + (ΔP/T_p(R))^2 + (GF/T_g(R))^2 )`

### 5-3. 判定（3段階）

* **GREEN**：`WCI < 0.7`
  → 続行OK（ただし10分ごと監視継続）
* **AMBER**：`0.7 ≤ WCI < 1.0`
  → 続行は可だが **(a)最寄り竜港へ収束 (b)高度をローカル層へ (c)積載が重いなら投棄/分割検討**
* **RED**：`WCI ≥ 1.0`
  → 続行NG。**着陸待機 / 引き返し / 安全空域へ退避**（優先）

---

## 6) “即わかる”例（850kg個体）

### ケースA：標準運用（積載120kg）

* `LR=0.141` → `R≈0.44` → 許容はまだある
* 10分で風が少し荒れた程度ならAMBERに留まりやすい

### ケースB：荷運び（積載200kg）

* `LR=0.235` → `R≈0.06` → 許容が激減
* 風の変化が同じでも、**Rが低い分WCIが跳ね上がる**
  → 「荷を積んだ瞬間、撤退判断が早くなる」＝狙いどおりの挙動

## 8) 離着陸フェーズ判定（LAモード）

### 8-1. LAモードに入る条件（どれか1つでON）

* **竜港管制圏内**（例：半径10km）
* **高度 200m AGL 以下**（最終進入・離陸直後の領域）
* **進入回廊／離陸回廊内**（ジオフェンス対象）
* **「最終進入（ファイナル）」コール後**
* **接地から離着陸帯クリアまで**（地上滑走＋脚洗浄導線に入るまで）

> 作中だと「ジオフェンスが鳴る」「回廊灯が見える」「管制と交信が増える」あたりがLAモードのスイッチ描写になります。

---

## 8-2. 目的：風の急変とガストを“指数で強く罰する”

巡航（上空）では吸収できる揺れでも、離着陸では

* 失速余裕が小さい
* 地面が近い
* 進入角が固定
  なので「同じΔでも危険度が上がる」を補正で表現します。

---

## 8-3. 補正方法（シンプルで運用しやすい）

通常の WCI 計算に、離着陸用重みを掛けた **LA-WCI** を使う。

### 通常WCI（復習）

* `WCI = sqrt( (ΔV_vec/T_w(R))^2 + (ΔP/T_p(R))^2 + (GF/T_g(R))^2 )`

### 離着陸用（LA-WCI）

* **風ベクトル変化の重み**：`k_w = 2.0`

* **ガスト幅の重み**：`k_g = 1.5`

* **気圧変化の重み**：`k_p = 1.0`（据え置き）

* `LA-WCI = sqrt( (k_w*ΔV_vec/T_w(R))^2 + (k_p*ΔP/T_p(R))^2 + (k_g*GF/T_g(R))^2 )`

> 直感：離着陸は「風の乱れ」と「突風」をより危険視する。

---

## 8-4. 判定基準（離着陸は“厳しめ”にする）

離着陸の閾値は、巡航より早く止める。

* **GREEN**：`LA-WCI < 0.6`
  → 進入/離陸 続行OK
* **AMBER**：`0.6 ≤ LA-WCI < 0.85`
  → 続行は可だが **速度/姿勢を保守的に**、必要なら **待機点へ戻る**
* **RED**：`LA-WCI ≥ 0.85`
  → **ゴーアラウンド（やり直し）／着陸延期／離陸中止**
  （L型は特に「一回やめてやり過ごす」が推奨）

> ここは“竜港運用の教範”として、操竜士に叩き込まれるルールにできます。

---

## 8-5. 実務的な“行動セット”（作中で映える）

### LA-WCIがAMBERなら

* **進入**：待機点に戻る／回廊灯追従／高度を一段上げて様子見
* **離陸**：風向が安定するまで待機／貨物があるなら分割・減載検討

### LA-WCIがREDなら（定型句に落とす）

* 管制へ：「**ゴーアラウンド**」 or 「**進入中止、待機**」
* 端末表示：`LA-WCI RED → ABORT / HOLD`
